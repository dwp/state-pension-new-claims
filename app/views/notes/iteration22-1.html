Business requirement

  • Agents need to be able to reissue a payment to the same bank account it was returned from

Scenario

  • A payment had been returned by the bank. Agents need to mark the payment as returned
  • Agent re-issues any returned payments
  • If agent doesn't change bank details they are asked if they want to pay into the same account where they are given the opportunity to change the bank details
  • If bank details have been changed the service will know and not ask this question
