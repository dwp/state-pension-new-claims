Request information and set a reminder

1. Request information from HMRC

Send an email to: ptopsca1741requestmailbox@hmrc.gov.uk requesting the Maxwell contributions for the 1991 to 1992 tax year.

Include in your email:

  • claimant name
  • claimant National Insurance number


2. Set a reminder in CAM

Set a reminder in CAM for 7 days to send another email if you do not get a reply.

When you receive the required information, you can return to the claimant's record and start the task again.
